# OS-widget

Displays a simple GUI that shows disk space remaining for a dual-booted/single-booted computer  
  
<img src="img/widget.png">


## Software tools:  
* Visual Studio Code  
* G++ 7.4 Compiler  
* gtkmm (C++ wrapper for gtk)  

## Dependencies:  
**Ubuntu/Debian**:
``` 
apt-get install libgtkmm-3.0-dev  
```
**Fedora**
```
yum install gtkmm30-docs
```

## Compile Instruction:  
```
make
```
