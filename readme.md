# OS-widget

Software tools:  
Visual Studio Code  
G++ 7.4 Compiler  
gtkmm (C++ wrapper for gtk)  
  
Compile Instruction:  
Use the 'make execute' command    
  
Displays a simple GUI that shows disk space remaining for a dual-booted computer
